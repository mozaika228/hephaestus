# Hephaestus Roadmap

## Q1 2026
- Stabilize web deployment on Render.
- Finalize OpenAI provider path and runtime observability.
- Improve API error reporting for provider failures.

## Q2 2026
- Complete media analysis pipeline for image, audio, and video.
- Add queue monitoring views in web UI.
- Add persistent storage migration path (SQLite -> Postgres optional).

## Q3 2026
- Ship desktop app packaging and update flow.
- Ship mobile app beta with push notifications.
- Add role-based access and audit trail basics.

## Q4 2026
- Expand integration catalog (Slack, Notion, Google Workspace).
- Add analytics dashboards for usage and model performance.
- Production hardening: retries, rate limiting, incident runbooks.

## Long-Term
- Multi-tenant workspaces.
- Provider policy routing by cost/latency.
- Agent workflows with human approval checkpoints.
